What is Responsive Design

Responsive web design is the process of designing web-based content intended for ease of accessibility across multiple browsers and devices.
Site navigation and browsing is designed to provide a user friendly interactive experience across different browser environments.
Also can be described as designing websites to be "mobile friendly".

Why Should I Be Concerned About Responsive Design

A significantly high percentage of web traffic is delivered to Mobile Devices
If my webpage is difficult to read, or even worse, unresponsive on any common browser environment, visitors will be deterred from viewing my content.

My List of Responsive Design Tips

  • Use relative units like percentages to scale content
  • Use exact dimensions for images to prevent loss of detail due to scaling
  • Test accessibility across all available platforms / browser environments
  • Begin planning web design with end-user ease of usage as a priority
