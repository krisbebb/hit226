What is a living style guide

A living style guide is a system used for documenting standards for user interfaces across an organisation.
These standards include a consistent colour scheme, patterns, logos, buttons and positioning across several related websites.
Living style guides also allow for common coding to be collected and referenced for reuse by developers.

Why is a living style guide useful to developers and designers

Living style guides are useful as they provide a means for consistency and standards during development and design of front-end user interfaces.
They are used as a way to document any changes or updates to interface styles or code being used.
Living style guides are an easy point of reference for new-starters, and guidelines for designers and developers to adhere to when styling interface experiences or code.
